API Parameters

Learn how to use the Amkyaw.Dev AI API

GET https://ai.amkyaw.workers.dev/?mode=chat&msg=Hello
Parameter Type Required Description
mode string Yes AI mode: chat, code, translate, summarize, roleplay
msg string Yes Your message or prompt (URL encoded)
lang string No Target language for translation (e.g., Japanese, Burmese)
persona string No Character for roleplay mode
Example Requests:
# Chat Mode
https://ai.amkyaw.workers.dev/?mode=chat&msg=Hello%20AI

# Code Generation
https://ai.amkyaw.workers.dev/?mode=code&msg=Python%20login%20form

# Translation (EN to Japanese)
https://ai.amkyaw.workers.dev/?mode=translate&lang=Japanese&msg=Hello

# Summarize
https://ai.amkyaw.workers.dev/?mode=summarize&msg=Long%20text%20here

# Roleplay
https://ai.amkyaw.workers.dev/?mode=roleplay&persona=teacher&msg=Hello
